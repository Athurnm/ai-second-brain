# No Em-Dash Enforcement

## Rule

**NEVER use the em-dash character (—) in any document, artifact, PRD, report, or any generated text.**

### What to Use Instead

| Instead of | Use |
|:---|:---|
| `—` (em-dash) | `-` (hyphen) or `--` (double hyphen) |
| `word—word` | `word - word` or `word -- word` |

### Examples

**Wrong:**
- `PIM Core — Product Information Management`
- `Q1 2026 — January through March`
- `The system — which handles all orders — must scale`

**Correct:**
- `PIM Core - Product Information Management`
- `Q1 2026 - January through March`
- `The system - which handles all orders - must scale`

### Scope

This rule applies to:
- All markdown documents (.md)
- All PRDs
- All reports and artifacts
- All Slack messages
- All Google Docs content
- Any text generated by the agent

### Verification

Before finalizing any document, search for the em-dash character (`—`) and replace all occurrences with a hyphen (`-`) or double hyphen (`--`).
