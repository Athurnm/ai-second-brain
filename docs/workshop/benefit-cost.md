# Buy Separately vs Build Your Own: The Cost Math

> Companion page for the AI Second Brain workshop. Numbers are list prices as of July 2026 and an assumed rate of Rp 16,300/USD. Prices change; the shape of the argument does not.

## The typical "AI stack", bought as separate subscriptions

| Tool | What it does | List price / month |
| :--- | :--- | ---: |
| ChatGPT Plus | General assistant | $20 |
| Otter.ai Pro | Meeting transcription and notes | $17 |
| Fireflies Pro | Meeting action items | $18 |
| Zapier Pro | Automation between apps | $20 |
| Notion AI | Docs and knowledge base AI | $10 |
| **Total** | | **$85 ≈ Rp 1,385,000** |

Three structural problems, beyond the price:

1. **The tools do not know each other.** Your meeting notes tool cannot see your todo list. Your automation tool cannot read your documents. Every tool holds a fragment of your context, and you are the integration layer.
2. **Each tool is someone else's roadmap.** You get the features they ship, at the pace they ship them, in the shape they decide.
3. **The count only grows.** Every new need means another subscription, another login, another silo.

## The second-brain approach

| Item | What it covers | Price / month |
| :--- | :--- | ---: |
| Claude Pro | The model + Claude Code agent. Runs every skill in this repo: meeting notes, reports, drafts, automation, document generation | $20 ≈ Rp 326,000 |
| AI Circle membership | Community, monthly live session, growing skill library, help when you are stuck | Rp 99,000 |
| **Total** | | **≈ Rp 425,000** |

**Savings: roughly Rp 960,000 per month (~69%)**, against the five-tool stack above.

The structural differences matter more than the savings:

1. **One context.** Your meetings, todos, documents, and drafts live in one workspace the AI reads together. The report knows what the meeting decided.
2. **Tools on demand.** A "tool" here is a markdown file plus, at most, a small script. When a new need appears, you build the skill in an afternoon instead of buying another subscription.
3. **You own it.** Local files, open template, no vendor deciding what your workflow looks like.

## Honest caveats

- Claude Pro has usage limits; very heavy daily use may want the higher tier ($100/mo), which still undercuts a grown separate-tool stack.
- Level 1+ connectors need a one-time setup effort (Google OAuth is the heaviest step, 1-2 hours).
- If you only ever need one of the five tools above, buying just that one is cheaper. The math favors the second brain as soon as your needs span two or more of them.
